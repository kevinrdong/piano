### Usage
點擊琴鍵可以彈奏相應的音符。

點擊 Sample 可將預設琴譜更改為另一琴譜。

點擊 Playnext 可播放琴譜上的次一音符。

點擊 Startplay 可連續播放紀錄之琴譜。

點擊 Startplay 後按鍵將變為 Stopplay ，點擊可終止正在播放的琴譜並回到琴譜起點。

點擊 Clear 可清除已紀錄之琴譜。

點擊 Startrecord 後按鍵將變為 Stoprecord ，在兩個動作之間彈奏的音符將會被記錄為新的琴譜。



含可彈奏的琴鍵、錄音與播放功能鍵，以及載入預設琴譜功能。
